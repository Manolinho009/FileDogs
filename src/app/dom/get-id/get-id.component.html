GetElementByID

Método utilizado para acessar um elemento por meio do seu ID, o qual é definido por meio do atributo id na parte do html.
EX: let elemento=document.getElementByID(‘input’)
Sendo:
Elemento: Usa referência ao element, ou pode ser ainda null se o id não foi especificado previamente.
ID:String que diferencia letras maiúsculas e minusculas e maiusculas, a qual e representada por um ID único do elemento desejado.
